
### 1Ô∏è‚É£ **–ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä–∏ (Dijkstra)**

#### üìå –û–ø–∏—Å:

- –í–∏–∑–Ω–∞—á–∞—î –Ω–∞–π–∫–æ—Ä–æ—Ç—à–∏–π —à–ª—è—Ö –≤—ñ–¥ –æ–¥–Ω—ñ—î—ó —Å—Ç–∞—Ä—Ç–æ–≤–æ—ó —Ç–æ—á–∫–∏ –¥–æ –≤—Å—ñ—Ö —ñ–Ω—à–∏—Ö.
- –ì–∞—Ä–∞–Ω—Ç–æ–≤–∞–Ω–æ –∑–Ω–∞—Ö–æ–¥–∏—Ç—å –æ–ø—Ç–∏–º–∞–ª—å–Ω–∏–π —à–ª—è—Ö, —è–∫—â–æ –≤—Å—ñ –≤–∞–≥–∏ —Ä–µ–±–µ—Ä –Ω–µ–≤—ñ–¥‚Äô—î–º–Ω—ñ (—â–æ —Ç–∞–∫ —ñ —î –≤ –¥–æ—Ä–æ–∂–Ω—å–æ–º—É –≥—Ä–∞—Ñ—ñ, –¥–µ –≤–∞–≥–∞ ‚Äî —Ü–µ –¥–æ–≤–∂–∏–Ω–∞ –¥–æ—Ä–æ–≥–∏ –∞–±–æ —á–∞—Å —É –¥–æ—Ä–æ–∑—ñ).
- –ü—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –ø–æ—à—É–∫—É –º–∞—Ä—à—Ä—É—Ç—É –º—ñ–∂ –¥–≤–æ–º–∞ —Ç–æ—á–∫–∞–º–∏.

#### ‚úÖ –ü–ª—é—Å–∏:

‚úî –ì–∞—Ä–∞–Ω—Ç—É—î –Ω–∞–π–∫–æ—Ä–æ—Ç—à–∏–π —à–ª—è—Ö.  
‚úî –ü—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –≤–µ–ª–∏–∫–∏—Ö –¥–æ—Ä–æ–∂–Ω—ñ—Ö –≥—Ä–∞—Ñ—ñ–≤.  
‚úî –î–æ—Å—Ç—É–ø–Ω—ñ –æ–ø—Ç–∏–º—ñ–∑–æ–≤–∞–Ω—ñ —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, —á–µ—Ä–µ–∑ **Priority Queue** –∞–±–æ **Fibonacci Heap**).

#### ‚ùå –ú—ñ–Ω—É—Å–∏:

‚úñ –ü—Ä–∞—Ü—é—î –ø–æ–≤—ñ–ª—å–Ω—ñ—à–µ –Ω–∞ –≤–µ–ª–∏–∫–∏—Ö –≥—Ä–∞—Ñ–∞—Ö, –Ω—ñ–∂ A*.  
‚úñ –ù–µ –ø—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –∑–º—ñ–Ω–∏ –º–∞—Ä—à—Ä—É—Ç—É –≤ —Ä–µ–∞–ª—å–Ω–æ–º—É —á–∞—Å—ñ.

#### üîπ –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ Python:

–ú–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ **networkx**, **scipy.sparse.csgraph** –∞–±–æ —Ä–µ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –≤—Ä—É—á–Ω—É —á–µ—Ä–µ–∑ **heapq**.

```python
import networkx as nx

G = nx.DiGraph()
G.add_weighted_edges_from([(1, 2, 10), (2, 3, 5), (1, 3, 20)])

path = nx.dijkstra_path(G, source=1, target=3, weight='weight')
print(path)  # –í–∏–≤–µ–¥–µ: [1, 2, 3]

```

### 2Ô∏è‚É£ **–ê–ª–≥–æ—Ä–∏—Ç–º A*** (A-star)

#### üìå –û–ø–∏—Å:

- –ú–æ–¥–∏—Ñ—ñ–∫–∞—Ü—ñ—è –î–µ–π–∫—Å—Ç—Ä–∏ –∑ –¥–æ–¥–∞–≤–∞–Ω–Ω—è–º –µ–≤—Ä–∏—Å—Ç–∏–∫–∏ (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –≤—ñ–¥—Å—Ç–∞–Ω—å –ø–æ –ø—Ä—è–º—ñ–π).
- –ü—Ä–∏—Å–∫–æ—Ä—é—î –ø–æ—à—É–∫ –∑–∞ —Ä–∞—Ö—É–Ω–æ–∫ –±—ñ–ª—å—à —Ü—ñ–ª–µ—Å–ø—Ä—è–º–æ–≤–∞–Ω–æ–≥–æ —Ä—É—Ö—É –≤ –±—ñ–∫ —Ü—ñ–ª—ñ.
- –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –≤ –Ω–∞–≤—ñ–≥–∞—Ü—ñ–π–Ω–∏—Ö —Å–∏—Å—Ç–µ–º–∞—Ö —Ç–∞ —ñ–≥—Ä–∞—Ö.

#### ‚úÖ –ü–ª—é—Å–∏:

‚úî –®–≤–∏–¥—à–∏–π –∑–∞ –î–µ–π–∫—Å—Ç—Ä—É –ø—Ä–∏ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–º—É –≤–∏–±–æ—Ä—ñ –µ–≤—Ä–∏—Å—Ç–∏–∫–∏.  
‚úî –•–æ—Ä–æ—à–∏–π –¥–ª—è –º–∞—Ä—à—Ä—É—Ç–∏–∑–∞—Ü—ñ—ó –≤ —Ä–µ–∞–ª—å–Ω–æ–º—É —á–∞—Å—ñ.

#### ‚ùå –ú—ñ–Ω—É—Å–∏:

‚úñ –ú–æ–∂–µ –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –≥—ñ—Ä—à–µ –Ω–∞ —Ä–µ–∞–ª—å–Ω–∏—Ö –¥–æ—Ä–æ–∂–Ω—ñ—Ö –∫–∞—Ä—Ç–∞—Ö —á–µ—Ä–µ–∑ —Å–∫–ª–∞–¥–Ω—ñ—Å—Ç—å –≤–∏–±–æ—Ä—É –µ–≤—Ä–∏—Å—Ç–∏–∫–∏.  
‚úñ –ù–µ –æ–ø—Ç–∏–º–∞–ª—å–Ω–∏–π –¥–ª—è –ø–æ—à—É–∫—É —à–ª—è—Ö—ñ–≤ –º—ñ–∂ –±–∞–≥–∞—Ç—å–º–∞ —Ç–æ—á–∫–∞–º–∏.

#### üîπ –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ Python:

```python
import networkx as nx

```

### 3Ô∏è‚É£ **–ê–ª–≥–æ—Ä–∏—Ç–º –§–ª–æ–π–¥–∞-–í–∞—Ä—à–∞–ª–ª–∞ (Floyd-Warshall)**

#### üìå –û–ø–∏—Å:

- –û–±—á–∏—Å–ª—é—î –Ω–∞–π–∫–æ—Ä–æ—Ç—à—ñ —à–ª—è—Ö–∏ –º—ñ–∂ **—É—Å—ñ–º–∞** –ø–∞—Ä–∞–º–∏ –≤–µ—Ä—à–∏–Ω.
- –ü—Ä–∞—Ü—é—î –∑–∞ $O(n^3)$, —â–æ –Ω–µ–ø—Ä–∏–π–Ω—è—Ç–Ω–æ –¥–ª—è –≤–µ–ª–∏–∫–æ—ó –¥–æ—Ä–æ–∂–Ω—å–æ—ó –∫–∞—Ä—Ç–∏ –£–∫—Ä–∞—ó–Ω–∏.

#### ‚úÖ –ü–ª—é—Å–∏:

‚úî –ú–æ–∂–µ –±—É—Ç–∏ –∫–æ—Ä–∏—Å–Ω–∏–º, —è–∫—â–æ —Ç—Ä–µ–±–∞ —à–≤–∏–¥–∫–æ –∑–Ω–∞—Ö–æ–¥–∏—Ç–∏ –º–∞—Ä—à—Ä—É—Ç–∏ –¥–ª—è –±–∞–≥–∞—Ç—å–æ—Ö –∑–∞–ø–∏—Ç—ñ–≤ –ø—ñ—Å–ª—è –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ–≥–æ –æ–±—á–∏—Å–ª–µ–Ω–Ω—è.

#### ‚ùå –ú—ñ–Ω—É—Å–∏:

‚úñ –ù–∞–¥—Ç–æ –ø–æ–≤—ñ–ª—å–Ω–∏–π –¥–ª—è —Ä–µ–∞–ª—å–Ω–æ–≥–æ –≥—Ä–∞—Ñ–∞ –¥–æ—Ä—ñ–≥ –£–∫—Ä–∞—ó–Ω–∏ (~–º—ñ–ª—å–π–æ–Ω–∏ –≤—É–∑–ª—ñ–≤).  
‚úñ –í–∏–º–∞–≥–∞—î –±–∞–≥–∞—Ç–æ –ø–∞–º‚Äô—è—Ç—ñ.

#### üîπ –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ Python:

```python
from scipy.sparse.csgraph import floyd_warshall

print(dist_matrix)
```

### 4Ô∏è‚É£ **–ê–ª–≥–æ—Ä–∏—Ç–º –ë–µ–ª–ª–º–∞–Ω–∞-–§–æ—Ä–¥–∞ (Bellman-Ford)**

### 6Ô∏è‚É£ **–ö–æ–º—ñ–≤–æ—è–∂–µ—Ä—Å—å–∫–∞ –∑–∞–¥–∞—á–∞ (TSP) + –î–µ–π–∫—Å—Ç—Ä–∞/A***

#### üìå –û–ø–∏—Å:

- –Ø–∫—â–æ —Ç—Ä–µ–±–∞ –∑–Ω–∞–π—Ç–∏ **–æ–ø—Ç–∏–º–∞–ª—å–Ω–∏–π —à–ª—è—Ö —á–µ—Ä–µ–∑ –∫—ñ–ª—å–∫–∞ —Ç–æ—á–æ–∫** (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, A ‚Üí B ‚Üí C ‚Üí D), —Ç–æ —Ü–µ –∑–∞–¥–∞—á–∞ –∫–æ–º—ñ–≤–æ—è–∂–µ—Ä–∞.
- –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å –∞–±–æ **–∂–∞–¥—ñ–±–Ω—ñ –∞–ª–≥–æ—Ä–∏—Ç–º–∏** (–Ω–∞–±–ª–∏–∂–µ–Ω—ñ —Ä—ñ—à–µ–Ω–Ω—è), –∞–±–æ **—Ç–æ—á–Ω—ñ –º–µ—Ç–æ–¥–∏** (–≥—ñ–ª–∫–∏ —Ç–∞ –º–µ–∂—ñ, –¥–∏–Ω–∞–º—ñ—á–Ω–µ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è).

#### ‚úÖ –ü–ª—é—Å–∏:

‚úî –ó–Ω–∞—Ö–æ–¥–∏—Ç—å –æ–ø—Ç–∏–º–∞–ª—å–Ω–∏–π –º–∞—Ä—à—Ä—É—Ç –¥–ª—è –∫—ñ–ª—å–∫–æ—Ö —Ç–æ—á–æ–∫.

#### ‚ùå –ú—ñ–Ω—É—Å–∏:

‚úñ –í–∞–∂–∫–æ —Ä–æ–∑–≤‚Äô—è–∑–∞—Ç–∏ —Ç–æ—á–Ω–æ –¥–ª—è –≤–µ–ª–∏–∫–æ—ó –∫—ñ–ª—å–∫–æ—Å—Ç—ñ —Ç–æ—á–æ–∫.

#### üîπ –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ Python:


## **1Ô∏è‚É£ Contraction Hierarchies (CH) ‚Äî –Ü—î—Ä–∞—Ä—Ö—ñ—ó —Å—Ç–∏—Å–Ω–µ–Ω–Ω—è**

üìå **–°—É—Ç—å:**  
CH ‚Äî —Ü–µ –º–µ—Ç–æ–¥ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ—ó –æ–±—Ä–æ–±–∫–∏ –≥—Ä–∞—Ñ–∞ –¥–ª—è –ø—Ä–∏—à–≤–∏–¥—à–µ–Ω–Ω—è –ø–æ—à—É–∫—É –Ω–∞–π–∫–æ—Ä–æ—Ç—à–∏—Ö —à–ª—è—Ö—ñ–≤. –í—ñ–Ω –ø—Ä–∞—Ü—é—î —à–ª—è—Ö–æ–º **"—Å—Ç–∏—Å–∫–∞–Ω–Ω—è" –º–∞–ª–æ–∑–Ω–∞—á—É—â–∏—Ö –≤—É–∑–ª—ñ–≤**, –∑–∞–ª–∏—à–∞—é—á–∏ –æ—Å–Ω–æ–≤–Ω—ñ –º–∞–≥—ñ—Å—Ç—Ä–∞–ª—ñ, —â–æ —Å—É—Ç—Ç—î–≤–æ –ø—Ä–∏—Å–∫–æ—Ä—é—î –ø–æ—à—É–∫.

üîπ **–Ø–∫ –ø—Ä–∞—Ü—é—î CH?**

1. **–ü–æ–ø–µ—Ä–µ–¥–Ω—è –æ–±—Ä–æ–±–∫–∞:**
    - –í—Å—ñ –≤—É–∑–ª–∏ —Å–æ—Ä—Ç—É—é—Ç—å—Å—è –∑–∞ –≤–∞–∂–ª–∏–≤—ñ—Å—Ç—é (–∞–≤—Ç–æ–º–∞–≥—ñ—Å—Ç—Ä–∞–ª—ñ –≤–∞–∂–ª–∏–≤—ñ—à—ñ –∑–∞ –º—ñ—Å—Ü–µ–≤—ñ –¥–æ—Ä–æ–≥–∏).
    - –ú–∞–ª–æ–∑–Ω–∞—á—É—â—ñ –≤—É–∑–ª–∏ **–ø—Ä–∏–±–∏—Ä–∞—é—Ç—å—Å—è (–∫–æ–Ω–¥–µ–Ω—Å—É—é—Ç—å—Å—è)**, –∞–ª–µ –¥–æ–¥–∞—é—Ç—å—Å—è "shortcut"-—Ä–µ–±—Ä–∞ –º—ñ–∂ –≤–∞–∂–ª–∏–≤–∏–º–∏ –≤—É–∑–ª–∞–º–∏.
2. **–ü–æ—à—É–∫:**
    - –ó–∞–ª–∏—à–∞—é—Ç—å—Å—è –ª–∏—à–µ –≤–∞–∂–ª–∏–≤—ñ –≤—É–∑–ª–∏, —â–æ —Å—É—Ç—Ç—î–≤–æ –ø—Ä–∏—Å–∫–æ—Ä—é—î –ø–æ—à—É–∫ —à–ª—è—Ö—ñ–≤.

‚úÖ **–ü–ª—é—Å–∏:**  
‚úî –î—É–∂–µ —à–≤–∏–¥–∫–∏–π –ø–æ—à—É–∫ (–≤ —Å–æ—Ç–Ω—ñ —Ä–∞–∑—ñ–≤ —à–≤–∏–¥—à–∏–π –∑–∞ –î–µ–π–∫—Å—Ç—Ä—É).  
‚úî –Ü–¥–µ–∞–ª—å–Ω–æ –ø—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –¥–æ—Ä–æ–∂–Ω—ñ—Ö –≥—Ä–∞—Ñ—ñ–≤, –æ—Å–æ–±–ª–∏–≤–æ –Ω–∞ —Ä—ñ–≤–Ω—ñ –∫—Ä–∞—ó–Ω–∏.  
‚úî –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –≤ OpenStreetMap (OSRM, GraphHopper).

‚ùå **–ú—ñ–Ω—É—Å–∏:**  
‚úñ –í–∏–º–∞–≥–∞—î **–∑–Ω–∞—á–Ω–æ—ó –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ—ó –æ–±—Ä–æ–±–∫–∏** (–º–æ–∂–µ –∑–∞–π–º–∞—Ç–∏ –≥–æ–¥–∏–Ω–∏ –Ω–∞ –≤–µ–ª–∏–∫–æ–º—É –≥—Ä–∞—Ñ—ñ).  
‚úñ –ù–µ –ø—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –¥–∏–Ω–∞–º—ñ—á–Ω–∏—Ö –∑–º—ñ–Ω (–ø–æ—Ç—Ä—ñ–±–Ω–æ –ø–µ—Ä–µ—Ä–∞—Ö–æ–≤—É–≤–∞—Ç–∏ —ñ—î—Ä–∞—Ä—Ö—ñ—é).

üîπ **–í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ Python:**

```python
import networkit as nk

# –ó–∞–≤–∞–Ω—Ç–∞–∂—É—î–º–æ –≥—Ä–∞—Ñ (—Ç—Ä–µ–±–∞ OSM-–¥–∞–Ω—ñ)
```

## 2Ô∏è‚É£ **ALT (A*, Landmarks, Triangle Inequality)**

üìå **–°—É—Ç—å:**  
ALT ‚Äî —Ü–µ –æ–ø—Ç–∏–º—ñ–∑–æ–≤–∞–Ω–∏–π –≤–∞—Ä—ñ–∞–Ω—Ç A*, —è–∫–∏–π –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î **–æ–ø–æ—Ä–Ω—ñ —Ç–æ—á–∫–∏ (landmarks)** —ñ **–Ω–µ—Ä—ñ–≤–Ω—ñ—Å—Ç—å —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫–∞** –¥–ª—è –∑–Ω–∞—á–Ω–æ–≥–æ —Å–∫–æ—Ä–æ—á–µ–Ω–Ω—è –∫—ñ–ª—å–∫–æ—Å—Ç—ñ –ø–µ—Ä–µ–≤—ñ—Ä–µ–Ω–∏—Ö –≤—É–∑–ª—ñ–≤.

üîπ **–Ø–∫ –ø—Ä–∞—Ü—é—î ALT?**

1. –í–∏–±–∏—Ä–∞—é—Ç—å—Å—è –∫—ñ–ª—å–∫–∞ **–æ–ø–æ—Ä–Ω–∏—Ö —Ç–æ—á–æ–∫** (landmarks) –Ω–∞ –≥—Ä–∞—Ñ—ñ (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –ö–∏—ó–≤, –õ—å–≤—ñ–≤, –•–∞—Ä–∫—ñ–≤).
2. –î–ª—è –∫–æ–∂–Ω–æ—ó –≤–µ—Ä—à–∏–Ω–∏ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ –æ–±—á–∏—Å–ª—é—é—Ç—å—Å—è **–≤—ñ–¥—Å—Ç–∞–Ω—ñ –¥–æ —Ü–∏—Ö —Ç–æ—á–æ–∫**.
3. –ü—ñ–¥ —á–∞—Å –ø–æ—à—É–∫—É –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –Ω–µ—Ä—ñ–≤–Ω—ñ—Å—Ç—å —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫–∞: $d(A, B) \leq |d(A, L) - d(B, L)|$ –¶–µ –¥–æ–ø–æ–º–∞–≥–∞—î –∑–Ω–∞—á–Ω–æ **–∑–≤—É–∑–∏—Ç–∏ –æ–±–ª–∞—Å—Ç—å –ø–æ—à—É–∫—É**.

‚úÖ **–ü–ª—é—Å–∏:**  
‚úî –®–≤–∏–¥—à–∏–π –∑–∞ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∏–π A*.  
‚úî –î–æ–±—Ä–µ –ø—Ä–∞—Ü—é—î –¥–ª—è –≤–µ–ª–∏–∫–∏—Ö –≥—Ä–∞—Ñ—ñ–≤, —è–∫—â–æ –æ–±—Ä–∞–Ω—ñ –ø—Ä–∞–≤–∏–ª—å–Ω—ñ –æ–ø–æ—Ä–Ω—ñ —Ç–æ—á–∫–∏.  
‚úî –õ–µ–≥—à–µ —Ä–µ–∞–ª—ñ–∑—É–≤–∞—Ç–∏, –Ω—ñ–∂ CH.

‚ùå **–ú—ñ–Ω—É—Å–∏:**  
‚úñ –ü–æ–ø–µ—Ä–µ–¥–Ω—ñ –æ–±—á–∏—Å–ª–µ–Ω–Ω—è –º–æ–∂—É—Ç—å –∑–∞–π–Ω—è—Ç–∏ –±–∞–≥–∞—Ç–æ —á–∞—Å—É.  
‚úñ –í–∏–±—ñ—Ä –ø–æ–≥–∞–Ω–∏—Ö –æ–ø–æ—Ä–Ω–∏—Ö —Ç–æ—á–æ–∫ –∑–Ω–∏–∂—É—î –µ—Ñ–µ–∫—Ç–∏–≤–Ω—ñ—Å—Ç—å.  
‚úñ –ì—ñ—Ä—à–µ –ø—Ä–∞—Ü—é—î –Ω–∞ –¥—É–∂–µ –¥–∏–Ω–∞–º—ñ—á–Ω–∏—Ö –≥—Ä–∞—Ñ–∞—Ö.

üîπ **–í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ Python:**

```python
import networkx as nx
import random

G = nx.Graph()
# –î–æ–¥–∞—î–º–æ –≤–µ—Ä—à–∏–Ω–∏ —Ç–∞ —Ä–µ–±—Ä–∞...
landmarks = random.sample(G.nodes(), 5)  # –í–∏–ø–∞–¥–∫–æ–≤—ñ –æ–ø–æ—Ä–Ω—ñ —Ç–æ—á–∫–∏
def heuristic(u, v):
    return min(abs(nx.shortest_path_length(G, u, l)) - abs(nx.shortest_path_length(G, v, l)) for l in landmarks)

path = nx.astar_path(G, source=1, target=100, weight='weight', heuristic=heuristic)
print(path)

```

üìå **–í–∏—Å–Ω–æ–≤–æ–∫:**  
–Ø–∫—â–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ **—à–≤–∏–¥–∫–æ –∑–Ω–∞—Ö–æ–¥–∏—Ç–∏ –º–∞—Ä—à—Ä—É—Ç–∏ –Ω–∞ –≤–µ–ª–∏–∫—ñ–π –∫–∞—Ä—Ç—ñ**, –∞–ª–µ –±–µ–∑ —Å–∫–ª–∞–¥–Ω–æ—ó –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ—ó –æ–±—Ä–æ–±–∫–∏, **ALT** ‚Äî —Ö–æ—Ä–æ—à–∏–π –≤–∏–±—ñ—Ä.
